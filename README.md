# uni
                Color Converter App
This application is a color converter tool built with Qt. It allows users to convert and visualize colors in RGB, HSV, and XYZ formats, as well as input and output HEX color codes.
  
  Features:
  
\RGB Sliders: Adjust the Red, Green, and Blue components of a color.

\HSV Sliders: Adjust the Hue, Saturation, and Value of a color.

\XYZ Sliders: Adjust the X, Y, and Z components of a color.

\HEX Input: Enter a HEX color code to update the color.

\Color Picker: Choose a color using a QColorDialog.

\Color Display: Shows the current color visually.

        Getting Started
    Prerequisites
    
Qt: Make sure you have Qt installed on your system. You can download it from Qt's official site.

    Exe
    
To try the app - use release.rar, there is a .exe file, which will run an application

    Installation
    
Clone the repository:

bash

Copy
git clone https://github.com/yourusername/color-converter-app.git
cd color-converter-app
Build the project: Open the project in Qt Creator and build it, or use the following command in the terminal:
bash

Copy
qmake && make
Run the application: Execute the binary created in the build directory.
    Usage
    
\Select Colors: Drag the sliders or input HEX codes to change colors.

\Convert Colors: Use sliders or HEX input to convert between color models.

\Pick Colors: Click "Choose Color" to use the color picker dialog.
